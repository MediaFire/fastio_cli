# @vividengine/fastio-cli

Command-line interface for the [Fast.io](https://fast.io) cloud storage platform.

## Install

```bash
npm install -g @vividengine/fastio-cli
```

Or run without installing:

```bash
npx @vividengine/fastio-cli --help
```

## Quick Start

```bash
# Authenticate
fastio auth login

# List your organizations
fastio org list

# Upload a file
fastio upload file --workspace <ID> ./myfile.txt
```
